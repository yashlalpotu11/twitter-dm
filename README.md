# Twitter DM Schedular

## Local setup

-   Clone the repo
    ```bash
     git clone https://github.com/yashlalpotu11/twitter-dm.git
    ```
-   Install dependencies
    ```bash
    cd twitter-dm
    npm install
    ```
-   Enter `recipientHandle`, `message` and `scheduleTime` in `index.js`
-   Run the script
    ```bash
    node index.js
    ```
